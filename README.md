#  Campus Shelf

Campus Shelf is a frontend university library website built using **HTML, CSS, and JavaScript**. It is designed to provide students with a simple and interactive digital library experience where they can browse books, discover resources, track borrowed books, and explore library services through an intuitive interface.

## Features

* Browse and search books available in the library.
* Bookmark books for future reading.
* Digitally track borrowed books and their due dates.
* Calculate overdue library fines using the built-in fine calculator.
* Login and submit library card details to access a personalized student profile.
* View borrowed books and achievement badges based on borrowing activity.
* Explore senior book recommendations.
* Read useful academic articles and access PDF resources.
* Discover new arrivals and library announcements.
* Responsive design for different screen sizes.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Open Library API
